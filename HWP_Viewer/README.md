# HWP 파일 뷰어

브라우저에서 `.hwp` 파일의 기본 정보, 텍스트 미리보기, 전체 텍스트/HEX 보기, Markdown 변환을 제공하는 간단한 뷰어입니다.

## 사용 방법

1. 이 폴더에서 정적 서버를 실행합니다.
   ```bash
   python -m http.server 8000
   ```
2. 브라우저에서 `http://localhost:8000`에 접속합니다.
3. `.hwp` 파일을 드래그하거나 클릭해서 업로드합니다.

## 참고 사항

- HWP는 바이너리 형식이므로 텍스트 추출은 제한적입니다.
- 파일은 브라우저에서만 처리되며 업로드되지 않습니다.
- Markdown 변환은 추출된 텍스트를 기반으로 간단히 구성됩니다.
