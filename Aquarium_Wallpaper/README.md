# 움직이는 수족관 바탕화면

캔버스 기반으로 기포가 떠오르고 빛이 비치는 수족관 배경을 보여주는 간단한 바탕화면입니다.

## 사용 방법

1. 이 폴더로 이동합니다.
2. 간단한 로컬 서버를 실행합니다.
   ```bash
   python -m http.server 4173
   ```
3. 브라우저에서 `http://127.0.0.1:4173/` 로 접속합니다.

## 구성

- `index.html`: 전체 화면 캔버스와 스크립트 연결.
- `styles.css`: 전체 화면 배경과 캔버스 스타일.
- `script.js`: 기포/플랑크톤 애니메이션, 조명 효과, 반응형 처리.
